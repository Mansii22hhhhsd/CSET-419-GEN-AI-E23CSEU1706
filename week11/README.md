# 🧪 Lab 11: Fine-Tuning GPT-2 for Real-World Applications

## 📌 Course
CSET419 – Introduction to Generative AI

---

## 🎯 Objective
The objective of this lab is to fine-tune a pre-trained generative model (GPT-2) for real-world applications.  
We adapt GPT-2 to two domains:
- Product Review Generation (E-commerce)
- Recipe Instruction Generation (Food-Tech)

---

## 🧠 What is Fine-Tuning?
Fine-tuning is the process of taking a pre-trained model and training it further on domain-specific data to improve performance on a specific task.

---

## 🎓 Learning Outcomes
After completing this lab, you will be able to:
- Load and configure GPT-2 using Hugging Face Transformers
- Prepare datasets for language modeling
- Fine-tune a model on custom data
- Compare outputs before and after training
- Evaluate generated text quality

---

# 🔹 Component I: Product Review Generator

## 📖 Problem Statement
Build a model that generates realistic product reviews for e-commerce platforms.

---

## ⚙️ Steps
1. Load GPT-2 model and tokenizer  
2. Generate baseline output (before training)  
3. Prepare product review dataset  
4. Fine-tune the model  
5. Generate and compare outputs
